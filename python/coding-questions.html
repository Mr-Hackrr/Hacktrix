Sum of Digits

Question: Write a Python program to find the sum of the digits of an integer.

How to Approach:

  • Understand the Problem: Sum the digits of a given integer.
  • Plan the Steps:
    • Take an integer as input.
    • Extract each digit and sum them up.
    • Print the sum.

Solution:

# Taking an integer as input
        num = int(input("Enter an integer: "))
        
        # Initializing sum
        sum_of_digits = 0
        
        # Loop to extract and sum the digits
        while num > 0:
            digit = num % 10
            sum_of_digits += digit
            num = num // 10
        
        # Printing the result
        print(f"Sum of the digits: {sum_of_digits}")

Explanation:

  • Input: The integer is taken as input.
  • Loop: A while loop is used to extract each digit using the modulus operator % and integer division //.
  • Sum: The digits are summed up and the result is printed.

Rotate a List

Question: Write a Python program to rotate a list to the right by a given number of steps.

How to Approach:

  • Understand the Problem: Rotate the elements of a list to the right by a specified number of steps.
  • Plan the Steps:
    • Take a list of integers as input.
    • Take the number of steps to rotate as input.
    • Use slicing to rotate the list.
    • Print the rotated list.

Solution:

# Taking a list of integers as input
numbers = [int(x) for x in input("Enter numbers separated by space: ").split()]

# Taking the number of steps to rotate as input
steps = int(input("Enter the number of steps to rotate: "))

# Rotating the list
steps = steps % len(numbers)  # To handle cases where steps > len(numbers)
rotated_list = numbers[-steps:] + numbers[:-steps]

# Printing the result
print(f"List after rotating by {steps} steps: {rotated_list}")

Explanation:

  • Input: The list of integers and the number of steps to rotate are taken as input.
  • Rotation: The list is rotated using slicing. The steps % len(numbers) ensures that the number of steps is within the bounds of the list length.
  • Output: The rotated list is printed.

Function to Calculate Factorial

How to Approach:

  • Understand the Problem: Calculate the factorial of a number using a function.
  • Plan the Steps:
    • Define a function that takes an integer as an argument.
    • Use a loop or recursion to calculate the factorial.
    • Return the factorial value.
    • Call the function and print the result.

Solution:

def factorial(n):
    """Calculate the factorial of a number."""
    if n == 0 or n == 1:
        return 1
    else:
        result = 1
        for i in range(2, n + 1):
            result *= i
        return result

# Taking an integer as input
num = int(input("Enter an integer: "))

# Calling the function and printing the result
print(f"Factorial of {num} is {factorial(num)}")

Explanation:

  • Function Definition: The factorial function is defined to calculate the factorial of a number.
  • Calculation: A loop is used to multiply the numbers from 2 to n.
  • Input and Output: The function is called with user input, and the result is printed.

Function to Check Prime Number

Question: Write a Python function to check if a given number is a prime number.

How to Approach:

  • Understand the Problem: Determine if a number is prime using a function.
  • Plan the Steps:
    • Define a function that takes an integer as an argument.
    • Check if the number is divisible by any number other than 1 and itself.
    • Return True if the number is prime, otherwise False.
    • Call the function and print the result.

Solution:

def is_prime(n):
    """Check if a number is prime."""
    if n <= 1:
        return False
    for i in range(2, int(n ** 0.5) + 1):
        if n % i == 0:
            return False
    return True

# Taking an integer as input
num = int(input("Enter an integer: "))

# Calling the function and printing the result
if is_prime(num):
    print(f"{num} is a prime number.")
else:
    print(f"{num} is not a prime number.")

Explanation:

  • Function Definition: The is_prime function is defined to check if a number is prime.
  • Prime Check: The function checks divisibility from 2 to the square root of n.
  • Input and Output: The function is called with user input, and the result is printed.
