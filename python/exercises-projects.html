Python List Operations Practice Questions

Python List Operations Practice Questions

1. Find the Maximum Element in a List

Write a function to find the maximum element in a list.

  • Test Case 1: max_element([1, 2, 3, 4, 5]) should return 5
  • Test Case 2: max_element([-10, -20, -30, -40]) should return -10

Hint: Use the built-in max() function.

2. Reverse a List

Write a function to reverse a list.

  • Test Case 1: reverse_list([1, 2, 3, 4, 5]) should return [5, 4, 3, 2, 1]
  • Test Case 2: reverse_list(['a', 'b', 'c']) should return ['c', 'b', 'a']

Hint: Use slicing.

3. Check if a List is Empty

Write a function to check if a list is empty.

  • Test Case 1: is_empty([]) should return True
  • Test Case 2: is_empty([1, 2, 3]) should return False

Hint: Use the len() function.

4. Count the Number of Occurrences of an Element in a List

Write a function to count the number of occurrences of an element in a list.

  • Test Case 1: count_occurrences([1, 2, 2, 3, 4, 2], 2) should return 3
  • Test Case 2: count_occurrences(['a', 'b', 'a', 'c', 'a'], 'a') should return 3

Hint: Use the count() method.

5. Remove All Occurrences of a Specific Element from a List

Write a function to remove all occurrences of a specific element from a list.

  • Test Case 1: remove_all([1, 2, 2, 3, 4, 2], 2) should return [1, 3, 4]
  • Test Case 2: remove_all(['a', 'b', 'a', 'c', 'a'], 'a') should return ['b', 'c']

Hint: Use a list comprehension.

6. Find the Second Largest Element in a List

Write a function to find the second largest element in a list.

  • Test Case 1: second_largest([1, 2, 3, 4, 5]) should return 4
  • Test Case 2: second_largest([10, 20, 4, 45, 99]) should return 45

Hint: Sort the list and access the second last element.

7. Merge Two Lists into One

Write a function to merge two lists into one.

  • Test Case 1: merge_lists([1, 2, 3], [4, 5, 6]) should return [1, 2, 3, 4, 5, 6]
  • Test Case 2: merge_lists(['a', 'b'], ['c', 'd']) should return ['a', 'b', 'c', 'd']

Hint: Use the + operator.

8. Find the Intersection of Two Lists

Write a function to find the intersection of two lists.

  • Test Case 1: intersection([1, 2, 3], [2, 3, 4]) should return [2, 3]
  • Test Case 2: intersection(['a', 'b', 'c'], ['b', 'c', 'd']) should return ['b', 'c']

Hint: Use list comprehension and the in keyword.

9. Find the Union of Two Lists

Write a function to find the union of two lists.

  • Test Case 1: union([1, 2, 3], [2, 3, 4]) should return [1, 2, 3, 4]
  • Test Case 2: union(['a', 'b', 'c'], ['b', 'c', 'd']) should return ['a', 'b', 'c', 'd']

Hint: Use the set data structure.

10. Flatten a Nested List

Write a function to flatten a nested list.

  • Test Case 1: flatten([[1, 2], [3, 4], [5]]) should return [1, 2, 3, 4, 5]
  • Test Case 2: flatten([['a', 'b'], ['c', 'd']]) should return ['a', 'b', 'c', 'd']

Hint: Use a nested loop or recursion.
